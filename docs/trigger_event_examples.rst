Trigger Events
===============

These are the events that eox_hooks listens to in order to trigger actions.

- ``post_register``
- ``post_session_login``
- ``post_certificate_creation``
- ``post_enrollment``
- ``post_unenrollment``
- ``post_enrollment_change``
- ``post_certificate_change``
- ``post_certificate_revoke``
- ``post_cohort_change``
- ``persistent_grade_summary_changed``
